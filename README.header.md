# TauErgon

This project is an experimental (coding) agent capable of generating and executing code.

## ⚠️ Important Notice

This software is provided "AS IS", without warranty of any kind.
Use at your own risk.

The agent may generate incorrect, unsafe, or incomplete code.
All outputs must be reviewed before use in any production or sensitive environment.

## Features

- Code generation
- Multi-step reasoning
- Tool use / execution (if applicable)
- Repository-aware assistance

## Safety Guidelines

Do NOT use this system:
- in production systems without review
- with sensitive credentials
- on systems you cannot afford to lose data from
- for malicious or unauthorized activities

Always run in a sandboxed environment.
